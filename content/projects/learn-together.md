A full-stack multiplayer learning application designed for German IT apprenticeship exam preparation (Fachinformatiker/IT-Systemelektroniker IHK Abschlussprüfung).

## 🎮 5 Game Modes

- **🃏 Flashcards** - Classic flip cards with progress tracking
- **❓ Quiz** - Multiple choice questions with scoring
- **⚔️ Duel** - Turn-based multiplayer (2-4 players, 20-second timer)
- **🏁 Race** - First correct answer wins, wrong answer locks you out
- **📝 Prüfung/Exam** - Realistic IHK exam simulation with 7 question types

## 📚 Content

- **107 Flashcards** across 3 topics
- **81 Quiz Questions** with explanations
- **20 Exam Questions** with German grading (1-6)

### Topics Covered
- 💾 Speichersysteme & Backup (Storage Systems)
- ☁️ Cloud Computing
- 🐳 Virtualisierung & Container (Virtualization)

## ⚡ Key Features

- **Real-time Multiplayer** - Socket.io powered Duel and Race modes
- **Server-Synced Progress** - Your progress saves automatically
- **Weak Cards Review** - Focus on cards you struggle with
- **Realistic Exam Mode** - 7 question types including matching, ordering, and calculations
- **German Grading System** - Authentic IHK-style scoring (1-6 scale)

## 🛠️ Tech Stack

- **Framework**: Next.js 14 (App Router)
- **Language**: TypeScript
- **Styling**: Tailwind CSS
- **Real-time**: Socket.io
- **Process Manager**: PM2
